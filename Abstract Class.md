# Abstract Class

Kendinden object instantiate edilmeyecek class. Bu classlar sadece diğer classlara extend edilmek için yazılırlar. İçlerinde abstract metodlar veya concrete metodlar olabilir. Bunların yanında değişken de tutabilirler. İçlerine yazılan abstract metodlar extend edildikleri subclasslarda override edilmek zorundalardır.
